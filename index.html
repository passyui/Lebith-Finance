📱 How to Sync Between Devices
Export on Device A
Set a passphrase and tap "Export Encrypted Backup". Share the .lbk file via WhatsApp, Google Drive, email, etc.
Open on Device B
Open Lebith Finance on your other device. Go to Settings → Device Sync → Import.
Import & Decrypt
Select the .lbk file, enter your passphrase, and your data will be restored securely.
Save the App
Use "Save Portable App" to save the HTML with all data embedded. Open it on any device — no install needed.
🛡 Security Information
🔐 AES-256 Encrypted ✓ Local Only ✓ No Cloud Server

All backup files are encrypted using AES-256 via the Web Crypto API. Your passphrase never leaves your device. Data integrity is verified with SHA-256 checksums on every import.
